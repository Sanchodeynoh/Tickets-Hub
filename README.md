# 🎟️ TICKETS HUB

An independent presale ticket resale marketplace for BTS, EXO, The Weeknd and more.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage — event listings, hero, how it works |
| `event.html` | BTS ARIRANG event detail — ticket tiers, buy now, add to cart |
| `register.html` | User registration |
| `login.html` | User login |
| `cart.html` | Shopping cart — order review + buyer details |
| `payment.html` | QR code payment page |
| `confirmation.html` | Order received confirmation |
| `admin.html` | Admin dashboard — manage orders & events |

## Hosting on GitHub Pages
1. Push all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/repo-name`
